# Design inspiration

A running list of sites and resources worth revisiting when thinking about the
look, feel, and structure of this blog. Notes on why each one is worth a look.

- **[Maggie Appleton](https://maggieappleton.com/)** — Illustrator and designer
  whose personal site is a widely-cited reference for the digital-garden format:
  growth stages, densely interlinked notes, and hand-drawn visual warmth. Very
  close in spirit to what this site is reaching for.
- **[Manuel Moreale](https://manuelmoreale.com/)** — A minimal, text-first
  personal blog with impeccable typography and restraint. Good model for keeping
  a site calm and readable with almost no chrome.
- **[NN/g — "Cards"](https://www.nngroup.com/articles/cards-component/)** —
  Nielsen Norman Group's UX guidance on the card component: when cards work, how
  to structure them, and common pitfalls. Useful reference for the post-list and
  curated-card layouts.
- **[TygoDesign](https://www.tygo.design/)** — A designer's portfolio/blog with a
  polished, considered aesthetic — handy for spacing, layout, and micro-detail
  ideas.
- **[Josh Comeau](https://www.joshwcomeau.com/)** — Front-end developer whose
  blog is a masterclass in delightful, interactive CSS: playful animations,
  strong typography, and thoughtful dark/light theming. Great for tasteful
  motion and polish ideas.
